# on-worldcup26-journal

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式存储，可直接通过浏览器访问，无需后端服务支持。仓库内容不针对任何特定网站或域名，仅作为页面存档与展示用途。

## 目录结构

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（如样式、脚本、图片等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- **pages/**：每个子文件夹或独立文件代表一个归档页面。
- **assets/**：公共资源目录，页面可引用其中的文件以保持一致性。
- **index.html**：仓库的入口页面，可用于导航或展示概览。

## 页面归档说明

- 所有页面均为纯 HTML 结构，兼容主流浏览器。
- 页面之间相互独立，不共享后端数据或动态内容。
- 归档内容可能随时间更新，但历史版本会保留在 Git 历史记录中。
- 如需引用或分发页面，请保留原始文件完整性。

## 维护说明

- 本仓库由个人维护，不定期更新和整理。
- 若需提交新页面或修正错误，欢迎发起 Pull Request。
- 所有提交的页面应遵循基本的 HTML 规范，避免引入外部恶意代码。
- 对于过时或不再需要的页面，可直接删除或移入 `archived/` 目录。
